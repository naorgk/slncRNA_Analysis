The code and datasets provided here are part of the paper:

<b>Synthetic liquid-liquid phase separated RNA-protein biocondensates reveal a bi-phasic cytosol in <i>E.coli</i></b>

Naor Granik, Noa Katz, Sarah Goldberg, Yoav Schechtman, Roee Amit

The datasets in the folder 'Trace files' comprise the experimental data collected for and discussed in the paper.
The Matlab code is designed to read, analyse and plot relevant figures from the paper. 

The script 'collect_statistics.m' works directly on the dataset and allows the user to save statistical information in the form of .MAT files

The scripts 'plot_statistical_data.m', 'poisson_fitting_plots.m' and 'quiescence_exponential_fit.m' work on the statistical data (files in 'Statistical files' folder).
